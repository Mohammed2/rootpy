rootpy.plotting.utils.canvases_with¶

rootpy.plotting.utils.canvases_with(drawable)[source]¶

Return a list of all canvases where drawable has been painted.

Note: This function is inefficient because it inspects all objects on all
canvases, recursively. Avoid calling it if you have a large number of canvases and primitives.
