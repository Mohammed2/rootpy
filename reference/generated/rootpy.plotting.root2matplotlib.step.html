rootpy.plotting.root2matplotlib.step¶

rootpy.plotting.root2matplotlib.step(h, logy=None, axes=None, **kwargs)[source]¶

Make a matplotlib step plot from a ROOT histogram.

Parameters:

h : Hist

A rootpy Hist

logy : bool, optional (default=None)

If True then clip the y range between 1E-300 and 1E300. If None (the default) then automatically determine if the axes are log-scale and if this clipping should be performed.

axes : matplotlib Axes instance, optional (default=None)

The axes to plot on. If None then use the global current axes.

kwargs : additional keyword arguments, optional

Additional keyword arguments are passed directly to matplotlib’s fill_between function.
Returns:

Returns the value from matplotlib’s fill_between function.
